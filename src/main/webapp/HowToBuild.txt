1. Clone repository
2. Go to the directory
3. Run "npm install" command
4. Above command will compile project and will add dependencies.


Notes : 
To add dependency in project, run below command :
npm i --save <dependancy name>
e.g. npm i --save jquery

To add jquery plugin, run below command :
npm i --save jquery <plugin name>
e.g. npm i --save jquery datatables.net
